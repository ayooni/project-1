Introduction to HTML

HTML is the standard markup language for creating Web pages, HTML stands for Hyper Text Markup Language

HTML describes the structure of a Web page

HTML Element

Tags: An HTML tag surrounds the content. All HTML tags must be enclosed within these < > brackets and every tag performs a different task. It is important to always mind the way you position your tags since a web browser reads an HTML document from top to bottom and left to right. HTML tags are used to create HTML documents and render their properties. Each HTML tag has different properties.

Elements: According to W3schools, An HTML element is defined by a start tag, some content, and an end tag. The HTML element is everything from the start tag to the end tag: for example, Content goes here...

Attributes: An attribute in HTML provides extra information about the element, and it is applied within the start tag. An HTML attribute contains two fields: name and value.

HTML Style CSS

HTML Style CSS

Cascading Style Sheets (CSS) is used to format the layout of a webpage. With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!
